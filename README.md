## Features

- Real-time messaging
- Typing indicator
- Peer-to-peer connection using WebRTC
- Built with Vite + React on the frontend and Node.js + Socket.io on the backend

---

## Run Locally

### Backend
```bash
cd backend
npm install
node server.js
```
### Frontend
```bash
cd ../frontend
npm install
npm run dev
```
